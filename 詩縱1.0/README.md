# 遊玩提示
常用詞.png可以給你一些唐詩常用詞的參考。我通常第一個都是猜 "魑魅魍魎釁" 因為筆畫比較多

# windows
要求：已經安裝python3.12並且已經加入到環境變數中
開啟方法：按下 windows-setup.bat，等(可能要幾一兩分鐘)，等錯誤訊息或是瀏覽器自己打開來(第一次開可能會失敗，F5可以解決)
